char identifica_alvo_atingido ( struct Mapa *  mapa,
int  i,
int  j,
FILE *  arquivo 
)

Imprime as coordenadas de um tiro e a mensagem correspondente ao efeito desse tiro. Se o tiro acerta uma embarcação, especifica-se o seu tipo. Além disso, atualiza na matriz a posição atingida pelo tiro, com um dos seguintes caracteres: '=' se o tiro atingiu a água; ∗ '*' se o tiro atingiu alguma embarcacão; '!' se o tiro atingiu o barco; '+' se o tiro atingiu alguma posicão do caminho percorrido pelo barco, exceto a posisão atual.

Parâmetros
mapaMapa alocado dinamicamente.
ii-ésima linha.
jj-ésima coluna.
Autor
Rodrigo, Thais
Parâmetros
arquivoArquivo para escrever os logs.
Retorna
Caractere correspondente ao alvo atingido.
