Definições dos tipos

typedef WINDOW* MASK

indica uma máscara de desenho. Uma máscara é uma imagem preto e branco, onde os pontos brancos correspondem a uma região transparente. Uma máscara está sempre associada a um PIC. Com isto é possível construir sprites, isto é, imagens com forma não retangular.

typedef WINDOW* PIC

corresponde a uma área de rascunho de desenhos. Você pode escrever, copiar ou desenhar em um objeto destes sem que suas modificações apareçam na tela. Depois você pode copiar o pedaço de interesse para uma janela ativa.

typedef struct Win WINDOW

Tipo básico de regiões desenháveis. Uma janela é um apontador para esse Tipo. (WINDOW*)
